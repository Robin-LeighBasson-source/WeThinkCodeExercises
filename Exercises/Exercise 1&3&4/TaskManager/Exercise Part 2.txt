Exercise Part 2: Feature Implementation Findings (Task Export to CSV)
1. Initial Search & Hypothesis formulation

Search Strategy: Instead of just searching for generic keywords like "export" or "csv", I looked for existing architectural patterns in the codebase, specifically the "Data Retrieval → Formatting → Output" flow.

Reference Pattern: I analyzed the stats and list commands. The list command was the perfect template because it already handles filtering by status and priority using existing TaskStorage methods.

2. Feature Mapping (Where the code will live)
To maintain the project's strict Separation of Concerns, the "Export to CSV" feature will be distributed across three specific layers:

Utility Layer (New File: utils/task_export.js): * Will contain a pure function tasksToCSV(tasksArray) responsible for taking an array of Task objects, extracting the headers (id, title, priority, status, etc.), formatting them into a comma-separated string, and escaping any commas inside task descriptions.

Business Logic Layer (app.js -> TaskManager): * Will get a new method: exportFilteredTasksToCSV(statusFilter, priorityFilter).

This method will call the existing this.listTasks() to retrieve the correct data, pass it to the tasksToCSV utility, and return the formatted string.

CLI Layer (cli.js): * Will get a new .command('export <filename>') block.

It will parse the command-line options (like -s todo), call taskManager.exportFilteredTasksToCSV(), and then use Node's native fs.writeFileSync() to save the resulting string to the user's hard drive.

3. Components Affected

cli.js (Modified to add the new command and file writing logic)

app.js (Modified to add the new bridge method)

storage.js (UNTOUCHED - the storage layer already has getAllTasks, getTasksByStatus, etc., and doesn't need to know about the CSV export).

4. Implementation Plan

Step 1: Create utils/task_export.js and write the CSV formatting logic. Write a Jest test to ensure it handles commas and line breaks in descriptions properly.

Step 2: Import the utility into app.js and write the exportFilteredTasksToCSV method.

Step 3: Update cli.js to register the export command using commander.js, pipe the flags to the TaskManager, and write the returned string to the file system using fs.
